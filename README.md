# Projeto de Aprendizagem de Máquina

## Requisitos

- Python 3.8
- pip

## Instalação

Este projeto utiliza `pip` para gerenciamento de dependências.

### Configurar ambiente

```bash
# Instalar dependências
pip install -r requirements.txt

# Verificar versão do Python
python3.8 --version
```

### Executar scripts

```bash
# Executar Python
python3.8 seu_script.py

# Executar Jupyter Notebook
jupyter notebook
```
